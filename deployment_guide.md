# Guide: How to host the app on Firebase

We use **Vite** (to build the app) and **Firebase** (for the database and hosting).

## Step 1: Create a Firebase Project

1.  Go to [console.firebase.google.com](https://console.firebase.google.com/ "null") and sign in with a Google account.
2.  Click **"Add project"** and name it `iron-and-shares`.
3.  Disable Google Analytics (not needed) and click **Create Project**.
    

## Step 2: Set up the Database & Auth

1.  **Authentication:**
    
    *   In the left sidebar, click **Build > Authentication**.
    *   Click **Get Started**.
    *   Select **Anonymous** from the Sign-in method list and **Enable** it. Click **Save**. _(This allows friends to join without creating accounts)._
        
2.  **Firestore Database:**
    
    *   In the sidebar, click **Build > Firestore Database**.
    *   Click **Create Database**.
    *   Choose a location (e.g., `nam5 (us-central)`).
        
    *   **Important:** When asked about security rules, choose **Start in test mode** (this allows anyone with the link to play for 30 days).
        
        *   _Note: For a permanent game, you'd configure specific rules, but Test Mode is fine for a quick game night._
            

## Step 3: Get Your Config

1.  Click the **Gear icon** (Project Settings) next to "Project Overview" in the sidebar.
2.  Scroll down to "Your apps" and click the **Web icon** (`</>`).
3.  Register the app (name it "IronShares").
4.  You will see a code block labeled `const firebaseConfig = { ... };`. **Keep this tab open**, you will need these values shortly.

## Step 4: Set up the Code on Your Computer

You need [Node.js](https://nodejs.org/ "null") installed on your computer.

1.  Open your terminal/command prompt.
2.  Run this command to create a new project:

        npm create vite@latest iron-shares -- --template react
        
3.  Move into the folder and install the necessary tools:
    
        cd iron-shares
        npm install firebase lucide-react    

## Step 5: Add the Game Code

1.  Open the project folder in a code editor (like VS Code).
2.  Open `src/App.jsx`.
3.  **Delete everything** in that file.
4.  **Paste** the entire code from the `OneFile.jsx` generated by the AI.

## Step 6: Fix the Configuration

In your new `src/App.jsx`, find these lines near the top:

    // --- Firebase Configuration ---
    const firebaseConfig = JSON.parse(__firebase_config); // <--- DELETE THIS LINE
    const appId = typeof __app_id !== 'undefined' ? __app_id : 'default-app-id'; // <--- DELETE THIS LINE
    
**Replace them** with the config you got in **Step 3**, like this:

    // --- Firebase Configuration ---
    const firebaseConfig = {
      apiKey: "AIzaSy...",
      authDomain: "iron-and-shares.firebaseapp.com",
      projectId: "iron-and-shares",
      storageBucket: "...",
      messagingSenderId: "...",
      appId: "..."
    };
    const appId = "my-public-game"; // You can name this whatever you want

## Step 7: Run and Deploy

1.  **Test locally:** Run `npm run dev` in your terminal. Open the `localhost` link to see if it works.    
2.  **Deploy to the World:** The easiest way to share it is using **Vercel** or **Netlify** (drag and drop), or strictly through Firebase Hosting:
    
    **Using Firebase Hosting (Recommended):**
    
    1.  In your terminal run: `npm install -g firebase-tools`
    2.  Run: `firebase login`
    3.  Run: `firebase init`
        
        *   Select **Hosting**.
        *   Select **Use an existing project** (choose `iron-and-shares`).
        *   What do you want to use as your public directory? Type: `dist`
        *   Configure as a single-page app? Type: `Yes`
            
    4.  Build your app: `npm run build`
    5.  Deploy: `firebase deploy`
        
Firebase will give you a URL (e.g., `https://iron-and-shares.web.app`). **Send this URL to your friends!**
